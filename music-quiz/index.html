One-time setup

To let players vote from their phones / laptops, we need a tiny free database. Takes ~3 minutes.

1 · Create a Firebase project

Free, no credit card required. The data lives in your account.

  1. Open console.firebase.google.com and click Add project. Name it anything (e.g. music-quiz). You can skip Google Analytics.
  2. On the project dashboard, click the </> Web icon to add a web app. Give it a nickname; you don't need Firebase Hosting.
  3. Firebase will show you a firebaseConfig object. Copy it.
  4. In the left sidebar: Build → Realtime Database → Create Database. Pick any location, then choose Start in test mode.

2 · Paste your Firebase config below

Stored only in your browser's localStorage. You'll only do this once on the host machine.

You can also paste raw const firebaseConfig = { ... }
